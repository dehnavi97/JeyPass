# JeyPass Secure Vault

![JeyPass Screenshot](https://raw.githubusercontent.com/dehnavi97/jeyPass/main/screenshot/main.png)

**JeyPass** is a modern, secure, and offline-capable password manager designed with simplicity and security in mind. Built with Next.js, Electron, and Capacitor, it provides a seamless experience across Windows and Android platforms.
