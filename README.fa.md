# صندوق امن جی‌پس 🇮🇷

![JeyPass Screenshot](https://raw.githubusercontent.com/dehnavi97/jeyPass/main/screenshot/fa.png)

**JeyPass** یک مدیر گذرواژه مدرن، امن و با قابلیت کار آفلاین است که با تمرکز بر سادگی و امنیت طراحی شده است. این برنامه از رمزگذاری قدرتمند (AES-GCM) برای محافظت از اطلاعات شما استفاده می‌کند و تضمین می‌کند که فقط شما با استفاده از گذرواژه اصلی خود به اطلاعاتتان دسترسی دارید. با استفاده از Next.js، الکترون و کپاسیتور، تجربه‌ای یکپارچه بر روی پلتفرم‌های **ویندوز، مک، لینوکس، اندروید و iOS** ارائه می‌دهد.

**لینک گیت‌هاب:** [https://github.com/dehnavi97/jeyPass](https://github.com/dehnavi97/jeyPass)

---

## ✨ ویژگی‌های کلیدی

- **کاملاً آفلاین:** بدون نیاز به اینترنت کار می‌کند. اطلاعات شما به صورت محلی و امن روی دستگاهتان ذخیره می‌شود.
- **رمزگذاری قدرتمند:** از الگوریتم AES-256-GCM برای رمزگذاری کل صندوق شما استفاده می‌کند. اطلاعات شما بدون گذرواژه اصلی غیرقابل خواندن است.
- **چند پلتفرمی:**
    - **دسکتاپ (ویندوز، مک، لینوکس):** به عنوان یک برنامه دسکتاپ نیتیو با استفاده از الکترون.
    - **موبایل (اندروید، iOS):** به عنوان یک برنامه موبایل نیتیو با استفاده از کپاسیتور.
- **رابط کاربری مدرن:**
  - رابط کاربری تمیز، زیبا و قابل سفارشی‌سازی با منوی شناور.
  - **تم‌ها:** شامل تم‌های روشن، تاریک، اقیانوس، کویر، یاقوت و جنگل برای شخصی‌سازی.
  - نوار عنوان سفارشی با قابلیت جابجایی و ذخیره وضعیت پنجره در نسخه دسکتاپ.
- **پشتیبانی از TOTP:** تولید و نمایش کدهای یکبار مصرف زمان‌بندی شده (احرازهویت دو مرحله‌ای).
- **مولد گذرواژه قوی:** گذرواژه‌های قوی، تصادفی و قابل تنظیم بسازید.
- **اشتراک‌گذاری امن:** اطلاعات ورود را به صورت امن از طریق QR Code به اشتراک بگذارید.
- **پشتیبان‌گیری و بازیابی:** به راحتی از صندوق رمزگذاری شده خود پشتیبان تهیه کرده و در صورت نیاز آن را بازیابی کنید (سازگار بین دسکتاپ و موبایل).
- **قفل خودکار:** دارای تایمر قفل خودکار با قابلیت توقف و ادامه برای افزایش امنیت.
- **پشتیبانی از چند زبان:** به طور کامل به زبان‌های فارسی، انگلیسی، عربی، فرانسوی، ترکی و آلمانی در دسترس است.

## 🛠️ تکنولوژی‌های استفاده شده

- **فریم‌ورک:** Next.js 14 (App Router)
- **دسکتاپ:** Electron
- **موبایل:** Capacitor
- **زبان:** TypeScript
- **استایل‌دهی:** Tailwind CSS
- **کامپوننت‌های UI:** ShadCN UI
- **چند زبانگی:** i18next
- **رمزگذاری:** Web Crypto API (AES-256-GCM)

## 🚀 شروع به کار

برای راه‌اندازی پروژه به صورت محلی، مراحل زیر را دنبال کنید.

### ۱. پیش‌نیازها

- [Node.js](https://nodejs.org/) (نسخه ۱۸ یا بالاتر)
- npm یا yarn
- برای توسعه اندروید: [Android Studio](https://developer.android.com/studio)
- برای توسعه iOS: [Xcode](https://developer.apple.com/xcode/) و [CocoaPods](https://cocoapods.org/)

### ۲. کلون کردن ریپازیتوری

```bash
git clone https://github.com/dehnavi97/jeyPass.git
cd jeyPass
```

### ۳. نصب وابستگی‌ها

```bash
npm install
```

### ۴. اجرا در حالت توسعه

- **برای الکترون (دسکتاپ):**
  برای اجرای برنامه در حالت توسعه با قابلیت بارگذاری مجدد سریع:
  ```bash
  npm run dev:electron
  ```

- **برای اندروید:**
  1. پروژه وب را با پروژه اندروید همگام‌سازی کنید:
     ```bash
     npm run sync:android
     ```
  2. پروژه اندروید را در Android Studio باز کنید:
     ```bash
     npm run open:android
     ```
  3. در Android Studio، برنامه را روی شبیه‌ساز یا دستگاه متصل اجرا کنید.

- **برای iOS:**
  1. پلتفرم iOS را اضافه کنید (اگر از قبل وجود ندارد):
     ```bash
     npx cap add ios
     ```
  2. پروژه وب را با پروژه iOS همگام‌سازی کنید:
     ```bash
     npm run sync:ios
     ```
  3. پروژه iOS را در Xcode باز کنید:
     ```bash
     npm run open:ios
     ```
  4. در Xcode، برنامه را روی شبیه‌ساز یا آیفون متصل اجرا کنید.

## 📦 خروجی گرفتن برای تولید

### ۱. آماده‌سازی آیکون‌ها (قدم اول مهم)

1.  آیکون برنامه خود را با ابعاد **1024x1024** طراحی کرده و با نام `assets/icon.png` ذخیره کنید.
2.  (اختیاری) می‌توانید یک آیکون برای حالت تیره با نام `assets/icon-dark.png` و یک پس‌زمینه برای صفحه اسپلش با نام `assets/splash.png` ایجاد کنید.
3.  دستور زیر را اجرا کنید تا تمام اندازه‌های آیکون و صفحات اسپلش به صورت خودکار تولید شوند:
    ```bash
    npm run assets:generate
    ```

### ۲. خروجی برای دسکتاپ (Electron)

- **برای ویندوز:** (باید روی سیستم‌عامل ویندوز اجرا شود)
  ```bash
  npm run build:electron
  ```
  فایل نصب (`.exe`) در پوشه `release` قرار خواهد گرفت.

- **برای مک:** (باید روی سیستم‌عامل macOS اجرا شود)
  ```bash
  npm run build:electron
  ```
  فایل نصب (`.dmg`) در پوشه `release` قرار خواهد گرفت.

- **برای لینوکس:**
  ```bash
  npm run build:electron
  ```
  فایل نصب (`.AppImage`) در پوشه `release` قرار خواهد گرفت.

### ۳. خروجی برای اندروید (Capacitor)

1.  **همگام‌سازی پروژه:**
    ```bash
    npm run sync:android
    ```
2.  **باز کردن در Android Studio:**
    ```bash
    npm run open:android
    ```
3.  **ساخت APK امضا شده:**
    - در Android Studio، به مسیر `Build > Generate Signed Bundle / APK...` بروید.
    - گزینه **APK** را انتخاب کرده و طبق دستورالعمل‌ها، یک "Key store" بسازید تا اپلیکیشن خود را امضا کنید.
    - پس از اتمام، فایل `app-release.apk` در مسیر `android/app/release` در دسترس خواهد بود.

### ۴. خروجی برای iOS (Capacitor)

1.  **همگام‌سازی پروژه:**
    ```bash
    npm run sync:ios
    ```
2.  **باز کردن در Xcode:**
    ```bash
    npm run open:ios
    ```
3.  **آرشیو و توزیع برنامه:**
    - در Xcode، به عنوان هدف `Any iOS Device (arm64)` را انتخاب کنید.
    - به مسیر `Product > Archive` بروید.
    - پس از اتمام آرشیو، پنجره Organizer باز می‌شود. از آنجا می‌توانید برنامه خود را (برای تست یا انتشار در اپ استور) توزیع کنید.

### ۵. خروجی به عنوان برنامه وب پیش‌رونده (PWA)

پروژه از قبل به عنوان یک PWA پیکربندی شده است. پس از اجرای `npm run build`، تمام فایل‌های لازم در پوشه `dist` قرار می‌گیرند. شما می‌توانید این پوشه را روی هر هاست استاتیک (مانند Vercel، Netlify یا GitHub Pages) قرار دهید تا برنامه به عنوان یک وب اپلیکیشن در دسترس باشد.
